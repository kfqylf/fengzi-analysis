## 三、做成「真·软件」的三条路

### 路线 A：便携包（最快，1 天内）

**适合：** 现在就想让人用，能接受「首次需 Python」

- 继续用 ZIP + `峰子分析.bat`
- 可选：用 **Inno Setup** 做安装向导（把文件夹打进 setup.exe，仍调用 bat）

**优点：** 几乎不用改代码  
**缺点：** 不够「专业」，用户怕 Python

---

### 路线 B：桌面安装包 exe（推荐 v1 正式版，1～2 周）

**适合：** 对外称「峰子分析 Windows 版」

技术方案（常见组合）：

```
PyInstaller 打包 Python + Streamlit
    → 启动时自动打开本机浏览器访问 127.0.0.1:8501
    → 或用 pywebview 内嵌窗口（更像独立软件）
```

步骤概要：

1. 写 `launcher.py`：启动 streamlit 子进程 + 打开窗口/浏览器  
2. `pyinstaller --onefile --windowed launcher.py`  
3. 用 **Inno Setup** 生成 `峰子分析_Setup_1.0.0.exe`  
4. 安装包带 Logo、卸载程序、开始菜单项  

**优点：** 用户无需懂 Python  
**缺点：** 安装包较大（约 150～300MB）；杀毒软件可能误报（需代码签名改善）

---

### 路线 C：SaaS 网页版（适合以后收费）

**适合：** 订阅制、按月付费

- 部署到：**阿里云 / 腾讯云 / Railway / Streamlit Cloud**  
- 用户浏览器访问：`https://fengzi.yourdomain.com`  
- 以后加：登录、Stripe/微信支付、套餐限制  

**优点：** 易收费、易更新  
**缺点：** 需服务器成本；行情 API 费用；合规要求更高

## Reddit (r/algotrading, r/stocks, r/dataisbeautiful)

**Title:** I built a free US sector rotation dashboard — 17 sectors, 10 leaders each, multi-timeframe heatmaps

I've been tracking sector rotation in US equities and couldn't find a free tool that goes beyond single ETFs. So I built one.

**Fengzi Analysis** — a real-time sector rotation dashboard:

- **17 sectors** — All 11 GICS sectors + 6 thematic sectors (Semiconductors, Commercial Space, Aerospace & Defense, Software, Biotech, Nuclear Power)
- **10 leaders per sector** — Equal-weight composite index instead of relying on a single ETF, giving a more accurate picture
- **Multi-timeframe analysis** — 1d / 5d / 20d / 60d / YTD returns + relative strength vs. SPY
- **Advanced charts** — Heatmap, rotation scatter plot, treemap, radar chart, lollipop charts
- **Bilingual UI** — Chinese / English
- **Optional AI analysis** — DeepSeek AI integration for narrative summaries

**Try it now (free, no signup):**
https://fengzistockanalysis.streamlit.app/

**GitHub (open source):**
https://github.com/kfqylf/fengzi-analysis

Tech stack: Python / Streamlit / Plotly / yfinance

Built this as a personal tool but figured others might find it useful. Feedback and stars appreciated!

## Hacker News (Show HN)

**Title:** Show HN: Fengzi Analysis – Open-source US sector rotation dashboard

**Body:**

I built an open-source tool to visualize US equity sector rotation across 17 sectors (11 GICS + 6 thematic like semiconductors, space, nuclear power).

Instead of tracking single ETFs, each sector is represented by an equal-weight composite of 10 leader stocks, which I found gives a more accurate signal.

Features:
- Multi-timeframe heatmap (5d/20d/60d/YTD)
- Rotation scatter plot (momentum vs. trend)
- Treemap, radar charts, constituent drill-down
- Relative strength vs. S&P 500
- Optional DeepSeek AI narrative summaries
- Bilingual (English/Chinese)

Live: https://fengzistockanalysis.streamlit.app/
Source: https://github.com/kfqylf/fengzi-analysis

Stack: Python, Streamlit, Plotly, yfinance. No API key required for basic use.
